# THE FINALS visual / UI / brand design research

**Research date:** 2026-07-19  
**Confidence:** High for the game’s stated creative pillars and public brand-asset availability; medium for UI characteristics inferred from official pages and documented feature descriptions; deliberately **not asserted** for exact brand colour values or fonts.

---

## Executive reading

THE FINALS is not fundamentally a cyberpunk product. Embark frames it as a **Dynamism Shooter** and a spectacular, global game show: player agency, experimentation, physical interactivity, distinct playstyles, competition, and an entertaining broadcast-like presentation are the durable cues. The visual system should therefore feel **live, legible, kinetic, competitive, and seasonally adaptable**—not like a dark “hacker terminal.” [Embark: Dynamism Shooter](https://www.embark-studios.com/) · [Gustav Tilleby interview](https://gamingtrend.com/interviews/embarks-gustav-tilleby-answers-our-questions-about-the-finals/)

For a Rank Tracker, borrow those cues at the level of information design and motion: scoreboard clarity, tournament/stage hierarchy, energetic reveals, clear contrast, and a restrained seasonal accent. Do **not** borrow THE FINALS’ logos, wordmark, proprietary imagery, in-game UI exports, or a font merely because a community source claims it is the game font.

## 4. Companion-app direction: evoke, do not imitate

### Recommended design system

1. **Build a neutral “broadcast results” foundation.** Use a dark or light neutral surface chosen for accessibility, one season accent, and a semantic status palette that remains fixed across seasons. This echoes competition/broadcast energy without claiming a THE FINALS colour scheme.
2. **Make rank the hero and history the arena.** A large current-rank module can lead the page; below it, a compact timeline/chart and sortable match/RS ledger let people investigate the run. This applies Embark’s stated agency and informativeness to a tracker. [Ranked Leagues post](https://www.reachthefinals.com/patchnotes/ranked)
3. **Use display typography as an original interpretation.** Choose a licensed condensed grotesk or sharply structured sans for headings, paired with an excellent UI sans for stats. Alter the word rhythm, numeral styling, and logo treatment enough that it reads as Rank Tracker—not an unofficial game surface.
4. **Use motion as a result reveal.** Animate rank change, new personal best, and chart selection within roughly one quick interaction beat; preserve the final state and provide reduced-motion alternatives. This follows Embark’s motion/iconography emphasis without reproducing its animations. [Embark UX Artist brief](https://careers.embark-studios.com/jobs/8027882-ux-artist-consultant-games)
5. **Create a seasonal “stage package,” not a reskin.** A seasonal label, abstract texture, and single accent can signal a new chapter while data components remain stable. This fits the game’s seasonal variation and avoids copying named maps, sponsor marks, or key art. [THE FINALS home](https://www.reachthefinals.com/)

### Safe original visual vocabulary

- **Use:** original abstract spotlights, score strips, bracket/standing metaphors, segmented dividers, time-based reveal, bold numerical hierarchy, and generic competition symbols.
- **Avoid:** a triangle/letterform treatment that reads as the THE FINALS wordmark, official contestant imagery, game-map photography, official UI screenshots, sponsor/faction marks, announcer names, or “world’s greatest game show” copy.
- **Naming:** describe the app as an **unofficial companion / independent Rank Tracker** where legally appropriate. Do not imply Embark endorsement, partnership, or data affiliation. The official creator rules prohibit misrepresentation and require assets to follow Embark guidelines. [Creator Code of Conduct](https://www.reachthefinals.com/code-of-conduct)

## 5. Concrete recommendations and anti-patterns

### Do

1. Put **current rank, rank delta, season, and last-updated time** in one immediately scannable header.
2. Give each result an explicit outcome word plus icon and number (`PROMOTED +132 RS`, `DOWN 48 RS`), rather than relying on a green/red flash.
3. Use an **asymmetric editorial grid**: one dominant current-performance module; smaller, regular modules for trend, mode breakdown, and recent matches.
4. Make filters feel like a player toolkit: quick presets, visible active scope, instant update, and a reset path. This is the companion-app translation of player agency/dynamism, not a literal game UI copy. [Tilleby interview](https://gamingtrend.com/interviews/embarks-gustav-tilleby-answers-our-questions-about-the-finals/)
5. Reserve spectacle for transitions and milestone states; use calm, high-contrast data surfaces for the persistent tracker.

### Do not

- **Do not default to retro-futurist cyberpunk.** No cyan/magenta-on-black CRT scanlines, matrix rain, chromatic-aberration type, terminal prompt copy, or permanent “glitch” effects. THE FINALS’ core is physical dynamism and entertainment-show competition, while its official seasonal worlds deliberately vary. [Embark project page](https://www.embark-studios.com/) · [THE FINALS home](https://www.reachthefinals.com/)
- **Do not make every UI element animated.** Continuous jitter, flicker, count-ups, and screen shake damage data comprehension; motion should signal a state change and remain optional.
- **Do not turn rank data into decoration.** Avoid low-contrast microtype, icon-only standings, or chart labels hidden behind hover on a core dashboard.
- **Do not freeze one season’s look as “the brand.”** A current theme is a campaign layer; the product needs its own durable visual identity.
- **Do not use protected assets as atmosphere.** A logo in a header, ripped title font, reconstructed faction mark, or cropped official screenshot is still brand/asset use—not merely inspiration.

## Validation checklist for the redesign

- [ ] A first-time user can identify current rank, direction of change, period, and freshness of the data in under five seconds.
- [ ] Statuses remain understandable in grayscale, at 200% zoom, and with reduced motion.
- [ ] Every season-specific visual can be removed without breaking hierarchy or semantic colour meaning.
- [ ] The UI contains no THE FINALS logo, key art, screenshot, faction/sponsor mark, copied UI component, or unlicensed font.
- [ ] Any use of an official creator asset is separately reviewed against the current official kit and its terms.
